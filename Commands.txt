🔑 Authentication & Identity
    PASS – authenticate with password
    NICK – set nickname
    USER – set username
    
💬 Basic Communication

    JOIN – join a channel
    PRIVMSG – send a private message (to user or channel)
    PART – leave a channel (not explicitly listed but usually expected)
    QUIT – exit the server (also expected but not required)

🛠️ Channel Operator Commands (Modes)

    KICK – remove a user from a channel
    INVITE – invite a user to a channel
    TOPIC – set or get the topic of a channel
    MODE – change channel mode
        i – invite-only
        t – topic settable only by ops
        k – password-protected channel
        o – grant/remove operator status
        l – limit users
